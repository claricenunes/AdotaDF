# PatasDF

# Rede Solidária DF

Plataforma comunitária para divulgar serviços gratuitos e iniciativas sociais no Distrito Federal.

## Objetivo

Facilitar o acesso da população a serviços como:

- assistência psicológica
- apoio jurídico
- doações
- cursos gratuitos

## Tecnologias

- HTML
- CSS
- JavaScript
- Google Forms
- Google Sheets
